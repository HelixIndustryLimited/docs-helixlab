# 切换模式 {#mode-switch}

---

Ankh 支持 USB 有线模式，以及蓝牙 LE（BLE）无线模式。

:::tip

如果你不想使用无线模式，可以断开电池排线的连接，或关闭电池开关 <sup>[<Icon icon="fa-solid fa-compass" /> 怎么做？](/docs-ankh-r2/battery/battery-switch)</sup>。

:::

## 切换到 USB 有线模式 {#switch-to-usb-mode}

在任何情况下，只需用 USB 数据线把 Ankh 连接至电脑，即可切换到有线模式。详见 [USB](/docs-ankh-r2/dual-mode/usb) 页面。

## 切换到无线模式 {#switch-to-wireless-mode}

由于在无线模式下，必须通过电池为键盘供电，因此需要把电池开关推至 **ON** <sup>[<Icon icon="fa-solid fa-compass" /> 怎么做？](/docs-ankh-r2/battery/battery-switch)</sup>，当 USB 数据线**未连接**时，Ankh 会自动切换到无线模式。

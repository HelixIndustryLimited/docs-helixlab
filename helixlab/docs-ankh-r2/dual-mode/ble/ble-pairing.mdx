# BLE 配对 {#ble-pairing}

Ankh 支持 BLE 5.4，向下兼容 4.2/4.0，最多支持配对**一台**主机。

## BLE 配对步骤 {#ble-pairing-procedure}

<Tabs queryString="os-variant">
  <TabItem value="windows" label="Windows" default>

对于 <Icon icon="fa-brands fa-windows" /> Windows，有两种配对方式，分别是：

* [**Swift 配对**](/docs-ankh-r2/dual-mode/ble/ble-pairing#swift-pairing) <sup><Icon icon="fa-solid fa-thumbs-up" /> 推荐</sup>，在 Windows 10 版本 1803 中引进，是把蓝牙设备配对至 Windows 主机的最新方式。无需用户手动搜寻键盘，只要键盘处于搜寻范围内并准备好配对，Windows 就会主动推送配对通知。
* [**手动配对**](/docs-ankh-r2/dual-mode/ble/ble-pairing#manually-pairing)，手动搜寻键盘并配对。

:::info

在 Windows 上使用 Swift 配对，须满足以下条件：

* 系统至少为 Windows 10 版本 1803。
* 主机蓝牙控制器支持 Swift 配对。

本教程内容以 Windows 11 23H2 为例。更多关于 Swift 配对的信息，请前往 [Microsoft Learn](https://learn.microsoft.com/zh-cn/windows-hardware/design/device-experiences/bluetooth-swift-pair) 查看。

:::

### Swift 配对步骤 {#swift-pairing}

<p className="step-number-text"> I. </p> 

Windows 的 Swift 配对可能不是默认开启的，请按下方视频的步骤检查该功能是否已经开启：

<ReactPlayer playing controls height="100%" url={swift_pair_tuts_1} />

> <text className="quote-text"> <Icon icon="fa-solid fa-caret-right" /> 前往“设置”。  
<Icon icon="fa-solid fa-caret-right" /> 选择左侧边栏中的“<Icon icon="fa-brands fa-bluetooth" /> 蓝牙和其他设备”。  
<Icon icon="fa-solid fa-caret-right" /> 选择“设备”。  
<Icon icon="fa-solid fa-caret-right" /> 打开“显示使用‘迅速配对’进行连接的通知 <Icon icon="fa-solid fa-toggle-on" />”。如果你找不到此按钮，则可能意味着你的电脑不支持 Swift 配对。</text>

<p className="step-number-text"> II. </p>

 确认 Ankh 处于 BLE 模式 <sup>[<Icon icon="fa-solid fa-compass" /> 怎么做？](/docs-ankh-r2/dual-mode/mode-switch)</sup>，如果之前曾与其他设备配对，可按 <kbd>**fn**</kbd> + <kbd>**X**</kbd> 删除之前的配对。

<p className="step-number-text"> III. </p> 

如下方视频所示，你的屏幕右下角应该会推送通知，点击“连接”即可完成配对。如果多次尝试仍失败，请参考下文的手动配对步骤。

<ReactPlayer playing controls height="100%" url={swift_pair_tuts_2} />

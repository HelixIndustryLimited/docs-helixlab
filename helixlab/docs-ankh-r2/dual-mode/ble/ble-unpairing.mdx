#  BLE 删除配对 {#ble-unpairing}

确认 Ankh 处于 BLE 模式 <sup>[<Icon icon="fa-solid fa-compass" /> 怎么做？](/docs-ankh-r2/dual-mode/usb-ble-mode-switch)</sup>，然后按 <kbd>**.**</kbd> + <kbd>**num lock**</kbd> 删除配对。

:::info

在删除配对后，你可立即将 Ankh 配对到新的主机，如果在 30 秒内未成功配对到新的主机，那么 Ankh 会重新配对到被删除的主机。

:::

# 切换 USB/BLE 模式 {#usb-ble-mode-switch}

Ankh 支持两种连接模式：

有线模式： USB <Icon icon="fa-brands fa-usb" />

无线模式： BLE <Icon icon="fa-brands fa-bluetooth" style={{color: "#74C0FC",}} />。

## 切换到 USB 有线模式 {#switch-to-usb-mode}

在任何时候，用 USB 数据线把 Ankh 连接至电脑，即可切换到有线模式。

## 切换到 BLE 无线模式 {#switch-to-ble-mode}

当 USB 数据线**未连接**时，Ankh 会自动切换到 BLE 模式。在 BLE 模式下，键盘通过内置电池供电，因此需要把电池开关推至 **ON** <sup>[<Icon icon="fa-solid fa-compass" /> 怎么做？](/docs-ankh-r2/battery/battery-switch)</sup>。

# 固件 {#firmware}

## 主控固件及 VIA JSON {#firmware-of-mcu-and-via-json}

:::caution

使用 [VIA](/docs-ankh-r2/via/conect-via) 时，你的固件版本与 VIA json 版本必须对应，否则 VIA 不能识别你的键盘。我们建议访问 [via.helix.site](https://via.helix.site)，因为该网站无需上传 json（仅限 Helix Lab 的新产品）。

:::

固件的使用方法，详见 [主控固件更新](/docs-ankh-r2/firmware-update/mcu-firmware-update) 章节。

VIA json 的使用方法，详见 [连接 VIA](/docs-ankh-r2/via/conect-via) 章节。

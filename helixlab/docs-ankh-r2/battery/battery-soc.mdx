# 电量百分比 {#battery-soc}

## 查看电量百分比 {#check-battery-soc}

在蓝牙模式下，通常可以从系统的蓝牙连接页面获取电量信息。

:::tip

如果电量始终显示为 100%，则可能说明电池未安装或[电池开关](/docs-ankh-r2/battery/battery-switch)未打开。

如果电池已经安装并且电池开关已经打开，但电量仍显示为 100%，请拔出 USB 数据线，关闭电池开关然后重新打开。

:::

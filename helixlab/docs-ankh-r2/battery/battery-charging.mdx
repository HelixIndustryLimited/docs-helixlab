# 充电 {#battery-charging}

当电池开关推至 **ON** <sup>[<Icon icon="fa-solid fa-compass" /> 怎么做？](/docs-ankh-r2/battery/battery-switch)</sup>，并且 USB 数据线**已连接**时，Ankh 会为电池充电。

电池从完全亏电到充满电大约需要 3 小时。

:::info

充电至 90% 以上时，充电速度会变慢以延长电池使用寿命。

拔出 USB 数据线后的短时间内，电量百分比可能快速降低，这属于正常现象，因为电量百分比是根据电池电压推算得出的，而充电时电池电压会偏高。

:::

:::caution

在 RGB 矩阵打开的情况下，充电速度会变慢。

:::

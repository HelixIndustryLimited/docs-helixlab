# 固件 {#firmware}

## 主控固件及 VIA JSON {#firmware-of-mcu-and-via-json}

:::tip

<details>
  <summary> <Icon icon="fa-solid fa-file-circle-question" /> 查看当前固件版本</summary>

打开任何一个可以提供输入的软件，如记事本，在键盘上按下组合键 <kbd>**fn**</kbd> + <kbd>**V**</kbd>，键盘会输出一个数字或字母，代表固件版本**代号**。例如，代号为 0 意味着固件版本为 1.0.0。

</details>

<details>
  <summary> <Icon icon="fa-solid fa-file-circle-question" /> 固件后缀名的含义</summary>

`_ll`: **Low Latency**，用更短的消抖时间和更激进的消抖算法换取低延迟，低端或老化的轴体可能会出现双击现象。详见 [延迟](/docs-elysium/miscellaneous/latency) 章节。

`_u8`: **USB 8kHz**，启用 USB 8kHz 轮询率，可能有助于降低 USB 链路延迟，也可能导致电脑 CPU 占用率升高。

</details>

:::

:::caution

使用 [VIA](/docs-elysium/via/conect-via) 时，你的固件版本与 VIA json 版本必须对应，否则 VIA 不能识别你的键盘。我们建议访问 [via.helix.site](https://via.helix.site)，因为该网站无需上传 json（仅限 Helix Lab 的新产品）。

:::

固件的使用方法，详见 [主控固件更新](/docs-elysium/firmware-update/mcu-firmware-update) 章节。

VIA json 的使用方法，详见 [连接 VIA](/docs-elysium/via/conect-via) 章节。

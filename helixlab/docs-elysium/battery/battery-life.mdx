# 续航时间 {#battery-life}

:::note

本页面内容仅适用于 Elysium 热插拔版本。

:::

Elysium 标配的电池为 3500mAh，在 RGB 矩阵关闭的情况下能连续打字 370 小时，在 RGB 矩阵打开的情况下能连续打字 18 小时（均以默认灯效和亮度计算）。

在电池充满电后，Elysium 能够深度睡眠 7000 小时（约合 291 天）。

:::tip

Elysium 采取的电量计算方式中，评估锂电池最佳工作区间所占据的权重较大，此参数与锂电池长期健康度有关，因此电量数值较为保守。实际上，在 RGB 矩阵关闭的情况下，即使电量百分比只剩 1%，仍然能连续打字 120 小时，而在 RGB 矩阵打开的情况下能连续打字 7 小时。

:::

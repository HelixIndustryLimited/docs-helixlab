# 电池开关 {#battery-switch}

:::note

本页面内容仅适用于 Elysium 热插拔版本。

:::

---

电池开关用于打开或关闭电池的连接。

下图为 Elysium 热插拔版本 PCB，标为红色的元件是电池开关。

![](./img/top_bat.svg)

:::caution

下图中标为红色的元件**不是**电池开关，勿随意调节。

此开关仅在工厂编程时使用，日常使用时请将其档位保持在**中间**，否则会引起无法切换无线模式等问题。

![](./img/top_mode.svg)

:::

## 打开电池开关 {#turn-on-battery-switch}

若要在无线模式下使用键盘，或给电池充电，则必须打开电池开关。

如下图所示，将电池开关的操作杆推至上侧，即可打开电池开关。

![](./img/switch_dir.svg)

:::tip

如果你从不使用无线模式，可以关闭电池开关。

:::

# 电池开关 {#battery-switch}

:::note

本页面内容仅适用于 Elysium 热插拔版本。

:::

---

电池开关用于打开或关闭电池的连接。

下图为 Elysium 热插拔版本 PCB，标为红色的元件是电池开关。

![](./img/top_bat.svg)

:::caution

请勿随意调节下图中标为红色的元件，该元件**并非**电池开关。

此开关仅用于工厂编程，日常使用时请务必将其保持在中间档位。若档位不在中间位置，可能导致无法切换无线模式等问题。

* 当开关拨到上侧时，键盘将强制使用 BLE 模式；

* 当开关拨到下侧时，键盘将强制使用 2G4 模式；

* 只有当开关位于中间档位时，你才能通过按下 <kbd>**fn**</kbd> + <kbd>**1**</kbd> / <kbd>**fn**</kbd> + <kbd>**2**</kbd> 来切换无线模式。

![](./img/top_mode.svg)

:::

## 打开电池开关 {#turn-on-battery-switch}

若要在无线模式下使用键盘，或给电池充电，则必须打开电池开关。

如下图所示，将电池开关的操作杆推至上侧，即可打开电池开关。

![](./img/switch_dir.svg)

:::tip

如果你从不使用无线模式，可以关闭电池开关。

:::

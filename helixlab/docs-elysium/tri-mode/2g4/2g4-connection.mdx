# 2G4 连接 {#2g4-connection}

:::note

本页面内容仅适用于 Elysium 热插拔版本。

:::

---

2G4，亦称 2.4Ghz，是频段在 2.4 ~ 2.5Ghz 的一种私有无线协议。

要让 Elysium 工作在 2G4 模式，你需要：

* 把 2G4 接收器插入电脑主机的 USB 接口中。
* 长按 <kbd>**fn**</kbd> + <kbd>**2**</kbd> 3 秒左右，把 Elysium 切换到 2G4 模式。

:::info

2G4 接收器在出厂时已经与你 Elysium 配对过，因此无需配对即可使用。

该配对信息是加密且唯一的，意味着你的 2G4 接收器无法连接其它 Elysium，除非 [重新配对接收器](/docs-elysium/tri-mode/2g4/2g4-re-pairing) 。

:::

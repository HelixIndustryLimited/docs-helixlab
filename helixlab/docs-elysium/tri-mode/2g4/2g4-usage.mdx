# 使用 2G4 {#2g4-usage}

:::note

本页面内容仅适用于 Elysium 热插拔版本。

:::

---

2G4，亦称 2.4Ghz，是频段在 2.4 ~ 2.5Ghz 的一种私有无线协议。

要让 Elysium 工作在 2G4 模式，你需要：

* 把 2G4 接收器插入电脑主机的 USB 接口中。
* 把 Elysium 切换到 2G4 模式 <sup>[<Icon icon="fa-solid fa-compass" /> 怎么做？](/docs-elysium/tri-mode/mode-switch/#switch-to-wireless-mode)</sup>。

#  BLE 删除配对 {#ble-unpairing}

:::note

本页面内容仅适用于 Elysium 热插拔版本。

:::

---

确认 Elysium 处于 BLE 模式 <sup>[<Icon icon="fa-solid fa-compass" /> 怎么做？](/docs-elysium/tri-mode/mode-switch/#switch-to-wireless-mode)</sup>，然后按 <kbd>**fn**</kbd> + <kbd>**X**</kbd> 删除配对。

:::info

在删除配对后，你可立即将 Elysium 配对到新的主机，如果在 30 秒内未成功配对到新的主机，那么 Elysium 会重新配对到被删除的主机。

:::

### USB 有线模式 {#usb-mode}

在任何时候，用 USB 数据线把 Elysium 连接至电脑，即可切换到有线模式。

### BLE 删除配对 {#ble-unpairing}

* 长按 <kbd>**fn**</kbd> + <kbd>**1**</kbd> 3 秒左右，以切换到 BLE 模式，然后短按 <kbd>**fn**</kbd> + <kbd>**X**</kbd> 删除配对。

<details>
  <summary> <Icon icon="fa-solid fa-file-circle-question" /> 删除配对时，如果已经处于 BLE 模式，还需要长按 <kbd>**fn**</kbd> + <kbd>**1**</kbd> 3 秒吗？</summary>

    不需要。

</details>

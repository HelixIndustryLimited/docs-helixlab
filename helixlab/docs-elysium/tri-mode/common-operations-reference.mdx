# 常用操作速查 {#common-operations-reference}

:::note

本页面内容仅适用于 Elysium 热插拔版本。

:::

### 切换到 USB 有线模式 {#switch-to-usb-mode}

在任何时候，用 USB 数据线把 Elysium 连接至电脑，即可切换到有线模式。

### 切换到 BLE 无线模式 {#switch-to-ble-mode}

* <HoverLink
    href="/docs-elysium/battery/battery-switch"
    svgPath={require('!!url-loader!./img/switch_dir.svg').default}
    offsetX={15}
    offsetY={240}
  >
    打开电池开关
  </HoverLink>

* 长按 <kbd>**fn**</kbd> + <kbd>**1**</kbd> 3 秒左右。

* 如未完成主机配对，请参阅[BLE 配对](docs-elysium/tri-mode/ble/ble-pairing.mdx)章节进行操作。否则，Elysium 将自动尝试连接已配对的主机。

### 切换到 2G4 无线模式 {#switch-to-2g4-mode}

* <HoverLink
    href="/docs-elysium/battery/battery-switch"
    svgPath={require('!!url-loader!./img/switch_dir.svg').default}
    offsetX={15}
    offsetY={240}
  >
    打开电池开关
  </HoverLink>

* 把 2G4 接收器插入主机的 USB 端口。

* 长按 <kbd>**fn**</kbd> + <kbd>**2**</kbd> 3 秒左右。

### BLE 删除配对

* 长按 <kbd>**fn**</kbd> + <kbd>**1**</kbd> 3 秒左右，以切换到 BLE 模式，然后短按 <kbd>**fn**</kbd> + <kbd>**X**</kbd> 删除配对。

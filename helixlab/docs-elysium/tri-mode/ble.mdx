# BLE {#ble}

:::note

本页面内容仅适用于 Elysium 热插拔版本。

:::

---

蓝牙 <Icon icon="fa-brands fa-bluetooth" style={{color: "#74C0FC",}} /> 是一种无线数据传输标准。Elysium 支持蓝牙 5.2，向下兼容蓝牙 4.2/4.0。

要通过蓝牙使用键盘，必须将 Elysium 与你的主机配对。Elysium 最多支持配对**一台**主机。

## 配对 {#pairing}

<Tabs queryString="os-variant">
  <TabItem value="windows" label="Windows" default>

对于 <Icon icon="fa-brands fa-windows" /> Windows，有两种配对方式，分别是：

* **Swift 配对** <sup><Icon icon="fa-solid fa-thumbs-up" /> 推荐</sup>：在 Windows 10 版本 1803 中引进，是把蓝牙设备配对至 Windows 主机的最新方式。无需用户手动搜寻键盘，只要键盘处于搜寻范围内并准备好配对，Windows 就会主动推送配对通知。
* **手动配对**：手动搜寻键盘并配对，属于传统的配对方式。
### Swift 配对步骤 {#swift-pairing}

:::info

在 Windows 上使用 Swift 配对，须满足以下条件：

* 系统至少为 Windows 10 版本 1803。
* 主机蓝牙控制器支持 Swift 配对。

本教程内容以 Windows 11 23H2 为例。更多关于 Swift 配对的信息，请前往 [Microsoft Learn](https://learn.microsoft.com/zh-cn/windows-hardware/design/device-experiences/bluetooth-swift-pair) 查看。

:::

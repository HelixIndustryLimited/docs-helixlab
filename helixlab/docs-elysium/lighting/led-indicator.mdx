# 指示灯 {#led-indicator}

## Caps Lock 指示灯 {#capslock-indicator}

<Tabs queryString="pcb-variant">
  <TabItem value="hotswap" label="热插拔" default>

    当 caps lock 触发时，字母区 RGB 灯会根据大小写锁定状态变色。

    如果 RGB 矩阵被关闭，那么即使大小写状态改变也不会引起任何灯光变化。

  </TabItem>
  <TabItem value="soldered" label="焊接">
    
    当 caps lock 触发时，该键下方的一颗 RGB 指示灯会点亮。

    你可以通过 VIA 自定义此灯光的特效、颜色和位置。

  </TabItem>
</Tabs>

## Scroll Lock 指示灯 {#capslock-indicator}

<Tabs queryString="pcb-variant">
  <TabItem value="hotswap" label="热插拔" default>

    没有 scroll lock 指示灯。

  </TabItem>
  <TabItem value="soldered" label="焊接">
    
    没有 scroll lock 指示灯。

  </TabItem>
</Tabs>

<img src={'/img/numbers/n_6.svg'} style={{ verticalAlign: '-0.16em', width: '18px' }} /> &nbsp; 用软排线连接左右两片 PCB：拉开 4 个连接座的黑色卡扣，塞入软排线并确保推至最深处，然后推回黑色卡扣。

<img src={'/img/numbers/n_7.svg'} style={{ verticalAlign: '-0.16em', width: '18px' }} /> &nbsp; 将子母板连接线的一端连接至主板。

**请阅读下面的注意事项：**

:::caution

在连接排线时，务必留意方向。

步骤 <img src={'/img/numbers/n_6.svg'} style={{ verticalAlign: '-0.2em', width: '18px' }} /> 中的软排线，一面是金手指，另一面是凸起的薄片，在安装软排线时，**确保金手指朝向 PCB，你能看见的是有薄片的那面。**

此外，软排线无法承受重量，在拿起内胆时需要同时拿起两边。

步骤 <img src={'/img/numbers/n_7.svg'} style={{ verticalAlign: '-0.2em', width: '18px' }} /> 中的连接线具有防呆功能，难以插反，勿使用蛮力。



:::

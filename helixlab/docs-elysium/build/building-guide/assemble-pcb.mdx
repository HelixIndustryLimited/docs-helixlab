步骤 <img src={'/img/numbers/n_6.svg'} style={{ verticalAlign: '-0.2em', width: '18px' }} /> 中，软排线的一面是金手指，另一面是凸起的薄片，在安装软排线时，**确保金手指面对着 PCB，这样你能够看到的是有薄片的那面。**

此外，软排线无法承受重量，因此在拿起内胆时，需要同时拿起左右两边，以避免对软排线造成拉扯或损坏。

步骤 <img src={'/img/numbers/n_7.svg'} style={{ verticalAlign: '-0.2em', width: '18px' }} /> 中的连接线具有防呆功能，难以插反，勿使用蛮力。

:::

:::info

如果键盘的右半部分的某一排按键或灯光失灵，而左半部分正常，则通常是由于软排线未安装到位或损坏导致的。

其中，未安装到位的情况包括：

* 正反面颠倒。

* 金手指未能与连接器充分接触。

:::

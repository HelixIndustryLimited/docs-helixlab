# 全键无冲  {#nkro}

全键无冲允许同时按下多于六个按键。

<Tabs queryString="pcb-variant">
  <TabItem value="hotswap" label="热插拔" default>

下图所示的青绿色按键为全键无冲键。

![](./img/elysium_keycodes_nkro.svg)

按下 <kbd>**fn**</kbd> + <kbd>**N**</kbd> 可在六键无冲和全键无冲之间切换。

  </TabItem>
